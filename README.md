DATN-Elec

An online auction platform developed as a graduation thesis project, enabling users to create listings, participate in real-time bidding, and manage auction activities through a modern web interface.

Features
User authentication and authorization
Product listing and auction creation
Real-time bidding system with Socket.IO
Auction countdown and status tracking
Personalized product recommendation system
User dashboard and auction history
Responsive UI for desktop and mobile devices
Tech Stack
Frontend
React.js
Next.js
Tailwind CSS
Redux
Backend
Node.js
Express.js
RESTful API
Socket.IO
Database
MongoDB
Recommendation Service
Python
Machine Learning-based Recommendation Engine
Getting Started
# Install dependencies
npm install

# Start development server
npm run dev
