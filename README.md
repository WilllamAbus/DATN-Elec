# DATN-Elec
 
DATN-Elec

An online auction platform developed as a graduation thesis project, enabling users to create listings, participate in real-time bidding, and manage auction activities through a modern web interface.

Features
-User authentication and authorization
-Product listing and auction creation
-Real-time bidding system with Socket.IO
-Auction countdown and status tracking
-Personalized product recommendation system
-User dashboard and auction history
-Responsive UI for desktop and mobile devices
Tech Stack
-Frontend: React.js, Next.js, Tailwind CSS, Redux
-Backend: Node.js, Express.js, RESTful API, Socket.IO
-Database: Mongodb
-Recommendation Service: Python, Machine Learning-based recommendation engine
Getting Started
# Install dependencies
npm install

# Start development server
npm run dev
Objectives

The project aims to provide a secure, scalable, and intelligent online auction system where users can buy, sell, and bid on products efficiently. By integrating real-time communication through Socket.IO and a Python-based recommendation engine, the platform enhances user engagement and delivers a more personalized auction experience.
